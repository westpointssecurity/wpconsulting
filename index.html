Services

Risks Management and Control

It's the process to identify, analyze, evaluate and treat Risks of any type in order to mitigate its adverse effects. Its regular stages are: 1) Identify the Risks 2) Analyze the Risks 3) Risks Assessment 4) Treat the Risks 5) Monitor/Review

Maritime Security Audits

Maritime Security Auditors whom are IMO regulated, are assigned to perform Audits based on the ISPS Code compliance, it´s the aim of the experts to verify in situ the current Maritime Security status of Port facilities Vessels and others.

Companies Security Audits and Evaluations

Auditors for companies security services are assigned to perform their tasks based on a pre – existing Plan or on a first time Audit procedure, it´s the aim of the experts to verify in situ the current Company´s Security status and of its facilities.

Companies Security Plans

To elaborate and present a Security Evaluation based on the current security status of the Company considering the identified Risks, which leads to the development of a Plan in order to increase security through out established procedures.

Port Facility Security Evaluation

Port Facility Security Evaluation (PFSE). Maritime Security Experts IMO regulated, are assigned to perform the (PFSE) based on the compliance of the International Ship and Port Facility Security Code (ISPS – Code).

Ship Security Evaluation

Ship Security Evaluation (SSE). Maritime Security Experts IMO regulated, are assigned to perform the (SSE) based on the compliance of the International Ship and Port Facility Security Code (ISPS – Code).

Port and Ship Security Plans

Port Facility Security Plan (PFSP), Ship Security Plan (SSP). Maritime Security Experts IMO regulated, are assigned to perform the (PFSP and SSP) based on the compliance of the International Ship and Port Facility Security Code (ISPS – Code).

Embarked Security Officer (OPE)

Maritime Security Experts whom are IMO regulated, are assigned to perform Onboard Merchant and all types of Vessels based on the ISPS Code compliance, as advisors to the Master in order to better up the Security of the Ship in accordance with the Ship´s Security Plan.

ISPS Code Training

Maritime Security Experts whom are IMO regulated, are assigned to give introductory courses and training of the ISPS Code when required by Maritime facilities and Ships, making much easier to understand and to comply the ISPS Code.

General Security Trainings

Security Experts with years in the field, are assigned to give general Security courses and training when required by different companies and its personnel.

Airport Facility Security Audits, Evaluations and Plans

Security Experts with years in the field, are assigned to perform Security Audits, Evaluations and Plans on Airport Facilities in order to increase security through out established procedures and the identification and mitigation of vulnerabilities.

Personnel Security Evaluations

Security Experts with years in the field, are assigned to comply Personnel Security Evaluations with the help of several security platforms, tools and information available, in order to establish a full reliability profile. Polygraph tests, complied by experts with the use of the latest technology.
